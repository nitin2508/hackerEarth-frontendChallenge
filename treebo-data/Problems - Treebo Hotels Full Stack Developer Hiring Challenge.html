| Hotel Deals — a simplified hotel searching experience!

[Problem Statement]

Develop a pseudo web application which would let users search and list hotel deals conveniently.

[Minimum Requirement]

— Utilise Data dump (SQL) for details on code submission:

  • Data Parameters:

  • hotel_deals[] — { id, name, image, rating, link, actual_price, discount, location }

  • name AS text (name of the challenge)

  • image AS text (image url of the hotel)

  • rating AS float (rating of the hotel)

  • link AS text (link associated with respective hotel)

  • actual_price AS text (price per day, associated with hotel)

  • discount AS text (percentage discount on actual_price would given final price)

  • location AS text (address of respective hotel)

— Minimal design listing hotel details

* Use Javascript, jQuery, AngularJS, or any of your favourite JS framework to process backend RESTFul queries.

::Backend

— Parse data dump to your favourite data format.

* You are free to use any method to store dataset.

* i.e.:  MySQL/PostgreSQL, MongoDB, CouchDB, redis.

— Emphasis on functional backend RESTful service developed in your favourite backend framework; (i.e. Could be Python/Django, Javascript/Node, Java/Servlet or the one which you feels fits the best).

— Functionally interactive API designed with basic function to retrieve hotel deals data (JSON response).

  • List submissions (add paging, 6 results per page)# /list?page=?

  • Implement RESTFul service responding statistical analysis on data (records): /stats

  • Response should contains:

— Submit Screenshots, Source code, Demo URL & Deployment Instructions.

[Plus Point]

— Utilise client-side scripting and UI functionality to enhance design; Use custom design, font and icons to make web-app more user-friendly.

— Implement Search (by name, or location) feature.

  • [RESTFul API] Full text search on ‘name’ or location parameter: /search?query=?

— Awesome if you could manage to deploy/host this application somewhere with Demo URL!

  • Cloud Providers (to deploy your application on one of the free environment):

    • Digital Ocean (use promocode: DROPLET10)

    • AWS

    • Openshift

— Calculate final_price (via %discount on actual_price)

[Extra Work]

— A feature to Like hotels and maintain Total Likes (via HTML5 Localstorage or WebSQL)

— Implement Sort (by Hotel Rating, Price or Like) feature.

  • [RESTFul API] Filter result: /list?sortBy=?

— Use your imagination and add features which would make feature rich web app; we always admire creative coding.

— You may add portfolio comprising awesome work you have done on web applications.
